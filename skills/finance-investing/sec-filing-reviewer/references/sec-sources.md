# SEC Review Sources

## Preferred Source Order

1. Official SEC APIs and filing pages
2. Official filing HTML or XBRL artifacts
3. Derived third-party parsers or wrappers

## Useful Official Endpoints

- `data.sec.gov/api/xbrl/companyfacts/`
- SEC filing metadata and submission data
- EDGAR full-text search for recent filing discovery

Use wrappers for convenience, but keep the official SEC URL in your notes.
