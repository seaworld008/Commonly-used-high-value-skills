# Analysis, Synthesis & Output Templates

Purpose: Turn raw data into insight cards, personas, journey maps, usability findings, and final reports.
Contents: thematic analysis, affinity synthesis, insight cards, usability plan, persona/journey templates, report template.

## Thematic Analysis

1. Familiarize: review notes, transcripts, or recordings.
2. Code: segment meaningful units, preserving participant wording when useful.
3. Group: cluster similar codes into themes.
4. Review: test theme coverage and consistency.
5. Finalize: define themes, select evidence, connect back to research questions.

## Usability Test Plan Defaults

| Item | Default |
|------|---------|
| Method | Moderated remote usability testing |
| Duration | `45 minutes` |
| Participants | `5-8 users` |
| Completion rate target | `>80%` |
| Error tolerance | `<3` |
| SUS target | `>68` |
