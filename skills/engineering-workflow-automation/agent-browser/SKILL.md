# Agent Browser

当任务需要 **直接操作网页**（而不是只读 API 数据）时使用本技能。

## 安装

```bash
npx clawhub@latest install agent-browser
```

## 适用场景

- 打开网页并执行多步操作（登录、查询、提交表单）
- 用自然语言定位元素（如“点击登录按钮”）
- 导出截图、视频、PDF 或执行页面内 JavaScript
- 保存 Cookie / 会话状态，支持多实例隔离

## 使用建议

1. 先说明目标页面与预期结果。
2. 再拆成可验证步骤（导航 → 定位 → 操作 → 校验）。
3. 涉及账号时优先使用隔离会话与最小权限。
