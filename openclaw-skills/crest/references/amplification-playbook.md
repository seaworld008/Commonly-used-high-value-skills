# Amplification Playbook

**Purpose:** Cross-platform content repurpose flows, distribution strategy, and monetization models.
**Read when:** STRATEGY or CONTENT mode — user needs content plans, repurpose maps, or monetization guidance.

---

## Core Principle: Create Once, Distribute Many

Every piece of content should be designed for repurposing from the start. The highest-effort format (long-form video or deep article) becomes the source; lower-effort derivatives extend reach.

---

## Cross-Platform Repurpose Flow

### Primary Flow (Video-First)

```
YouTube Long-form (8-15 min)
  ├─→ YouTube Shorts (3-5 clips × 15-60 sec)
  ├─→ TikTok (reformat for vertical, add captions)
  ├─→ Instagram Reels (15-60 sec, add screen text)
  ├─→ X Thread (text summary, 5-10 tweets)
  ├─→ Tech Blog Article (Qiita/Zenn/own blog)
  └─→ note (story/context version for non-tech audience)
```

### Primary Flow (Article-First)

```
Deep Tech Article (Qiita/Zenn/own blog)
  ├─→ X Thread (key insights, 5-10 tweets)
  ├─→ LinkedIn Post (professional angle + link)
  ├─→ YouTube Short/TikTok (screen recording of key demo)
  ├─→ note (behind-the-scenes context and decision-making version)
  ├─→ Newsletter (curated summary + commentary)
  └─→ Conference CFP (expand into talk proposal)
```

### Primary Flow (Talk-First)

```
Conference Talk (30-60 min)
  ├─→ Blog Post (written version with extras)
  ├─→ Slide Publication (Speaker Deck)
  ├─→ YouTube (recording or re-record)
  ├─→ X Thread (top 5 takeaways)
  ├─→ Qiita/Zenn (technical deep-dive of one section)
  └─→ Shorts/Reels (1-min highlight clips)
```

## Content Pillar Strategy

### Define 3-5 Content Pillars

Pillars are recurring themes aligned with your niche. All content should map to a pillar.

```
Example for "Rust × FinTech × Performance" niche:

Pillar 1: Rust Performance Patterns (deep technical)
Pillar 2: FinTech System Architecture (domain expertise)
Pillar 3: Engineering Culture & Process (management perspective)
Pillar 4: Career & Learning Journey (personal story)
```

### Pillar Distribution

| Pillar Type | Content Mix | Platform Fit |
|-------------|-------------|-------------|
| Technical deep-dive | 40% | Qiita, Zenn, YouTube |
| Domain insights | 25% | Zenn, note, LinkedIn |
| Process/culture | 20% | note, LinkedIn, X |
| Personal journey | 15% | note, X, Instagram |

## Platform-Specific Publishing Strategy

### Qiita / Zenn / Blog — Timing

| Factor | Recommendation |
|--------|---------------|
| Best posting day | Tuesday-Thursday (weekday traffic peak) |
| Best time | 7:00-9:00, 12:00-13:00, 18:00-21:00 (JST) |
| Frequency | 2-4 articles/month (quality > quantity) |
| Seasonal peak | December (Advent Calendar) — plan from October |

### X (Twitter) — Posting Strategy

| Rule | Detail |
|------|--------|
| Frequency | 1-2 posts/day minimum |
| Thread timing | Morning (7-9 JST) or evening (19-21 JST) |
| Link placement | Always in reply, never in main tweet |
| Engagement ratio | 70% engaging with others, 30% own content |
| Thread structure | Hook tweet (no link) → content → CTA with link in final tweet |

### YouTube — Upload Strategy

| Rule | Detail |
|------|--------|
| Frequency | 1-2 long-form/month + 3-5 Shorts/week |
| Title formula | "[Number] + [Specific topic] + [Benefit/Curiosity]" |
| Thumbnail | 3-5 words, high contrast, face if possible |
| Description | First 2 lines = keyword-rich summary (above fold) |
| End screen | Subscribe + next video CTA in last 20 seconds |

## Monetization Models

### Revenue Source Mix ($100K+/year creators)

| Revenue Source | Typical Share | Key Factor |
|---------------|--------------|-----------|
| Sponsorship/Consulting | 40-60% | Niche audience relevance > scale |
| Membership/Subscription | 20-30% | Zenn Book, note magazine, Patreon |
| Ad Revenue | 10-20% | YouTube AdSense, blog ads |
| Affiliate | 5-10% | Tool/service recommendations |

**Rule:** 3+ revenue sources = 5× income of single-source creators.

### Japan-Specific Monetization

| Platform | Method | Typical Range |
|----------|--------|--------------|
| Zenn | Paid Book | ¥200-1,000/Book |
| Zenn | Badge (tips) | Variable |
| note | Paid Article | ¥100-1,000/article |
| note | Magazine Subscription | ¥500-2,000/month |
| note | Membership | ¥500-5,000/month |
| YouTube | AdSense | CPM ¥300-1,500 (tech niche) |
| YouTube | Channel Membership | ¥190-6,000/month |

### Monetization Readiness Checklist

| # | Criterion | Threshold |
|---|-----------|-----------|
| 1 | Consistent audience | 500+ followers on primary platform |
| 2 | Regular output | 3+ months of consistent publishing |
| 3 | Engagement signal | Regular comments/replies from audience |
| 4 | Clear niche | L2+ positioning established |
| 5 | Trust built | Free content demonstrates expertise |

**Rule:** Don't monetize before building trust. Free value → audience → monetization, never the reverse.

## Time Investment vs Return

### Platform Priority Matrix

| Priority | Platform | Weekly Hours | Payoff Timeline | Primary Return |
|----------|----------|-------------|----------------|---------------|
| 1 | X + Qiita/Zenn | 5-8h | 3-6 months | Network + SEO + portfolio |
| 2 | YouTube | 10-20h | 6-12 months | Revenue + authority |
| 3 | Shorts/TikTok/Reels | 3-5h | 1-3 months | New audience discovery |
| 4 | Instagram | 3-5h | 3-6 months | Visual brand |
| 5 | note | 2-3h | 3-6 months | Non-tech reach + revenue |

**Time-constrained recommendation:** Start with Priority 1. Add Priority 2 when you can sustain 10+ hours/week. Add others incrementally.

### ROI Optimization Rules

1. **Repurpose before creating** — check if existing content can be reformatted
2. **Batch production** — record/write multiple pieces in one session
3. **Template everything** — use consistent formats to reduce decision fatigue
4. **Automate distribution** — use scheduling tools for cross-posting
5. **80/20 engagement** — focus on platforms where your audience actually is
